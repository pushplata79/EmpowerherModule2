HTML is the standard markup language for creating Web pages.


Basic Structure of an HTML Document

1.Declaration
<!DOCTYPE html>:This tells the browser that the document is an HTML5 document. It should always be the first line.
2. HTML Root Tag
<html>:The root element of the HTML document. Everything inside this tag represents the content of the webpage.
3. Head Section
<head>:This section contains metadata about the webpage, which is not visible on the page itself. Common elements include:
•
<title>: Defines the title displayed in the browser tab.
•
<meta>: Provides metadata like character encoding (<meta charset="UTF-8">) or viewport settings.
•
<link>: Links external stylesheets or resources.
•
<style>: Embeds internal CSS.
•
<script>: Adds JavaScript.

  4. Body Section

<body>:Contains the visible content of the webpage, such as text, images, links, and other elements.

<p>This is a paragraph of text on my webpage.</p>
</body>
</html>



Explanation of Example
1.
<!DOCTYPE html>: Declares the document as HTML5.
2.
<html lang="en">: Starts the HTML document and sets the language to English.
3.
<head>: Contains the metadata and the title of the page.
4.
<meta charset="UTF-8">: Ensures the text is displayed correctly for most characters.
5.
<meta name="viewport" content="width=device-width, initial-scale=1.0">: Ensures the page is mobile-friendly.
6.
<title>: Displays "My First Webpage" in the browser tab.
7.
<body>: Contains the visible content—a heading and a paragraph
